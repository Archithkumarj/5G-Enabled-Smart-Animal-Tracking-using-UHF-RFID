# 5G-Enabled-Smart-Animal-Tracking-using-UHF-RFID
A real-time livestock monitoring system integrating 930 MHz UHF-RFID, Raspberry Pi 4, IoT Gateway, and 5G communication. Passive RFID tags with unique EPC are used to identify animals, while RSSI-based distance estimation enables location tracking. Data is transmitted through 5G gNodeB → Core Network → MEC Server → Web Server for low-latency monitoring and real-time updates.

Features
UHF RFID-based animal identification
RSSI-based distance estimation and tracking
Raspberry Pi 4 edge processing
5G-enabled low-latency communication
MEC server and web-based monitoring

Tech Stack: UHF-RFID | Raspberry Pi 4 | IoT | 5G | MEC | Python | Flask
